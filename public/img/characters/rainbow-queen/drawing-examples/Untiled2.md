# Rainbow Glow Effect Tutorial

This guide outlines the process of creating a vibrant, glowing spectral flower effect using layering, blurs, and masks.

### 1. Create the Base Lines

First, we make the core linework.

* **Action:** Create a clone of your line art.
* **Masking:** Apply a white background mask (**Group 2**).
* **Effect:** Apply a heavy **Motion Blur** to create the initial light spread.

### 2. Inner Glow and Coloring

Next, we add the chromatic element.

* **Action:** Create another clone of the lines.
* **Masking:** Use a colored/rainbow image as a mask (**Group 5**).
* **Effect:** Apply a **Motion Blur** with a smaller radius than the first step to maintain some detail while adding color.

### 3. Final Bloom and Depth

To unify the glowing effect and add photographic depth:

* **Action:** Merge the previous groups and place the result at the top of the layer stack.
* **Effect:** Apply a **Lens Blur** to simulate a realistic light "bloom" effect.

### 4. Core Rainbow Center

Finally, we add the soft central glow.

* **Action:** Create a circle with soft, feathered edges and place a rainbow gradient inside it (**Group 8**).
* **Recommended Blending Modes:**
| Tier | Blend Modes |
| :--- | :--- |
| **Primary** | Overlay, Addition |
| **Secondary** | Divide, Color Dodge, Gamma Illumination |
| **Special** | Luminosity / Shine (SAI) |
