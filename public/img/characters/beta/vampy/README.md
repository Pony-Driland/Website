# 🦄 Vampy

Welcome to the reference folder for **Vampy**! 

This character was originally designed for a different fanfic that never actually made it past the planning stage. It was supposed to be a story about the Dreasond family (which is the origin of my current username).

While some ideas were developed, the fic was never truly written. This character also served as a reference for Rayane's personality, but she had always been left abandoned in my drawing folder. Now, I'm bringing her here to try and build something completely new for her.

## 🎨 About the Design
This character is completely original and was 100% designed and created by me.
