# 🦄 Luana

Welcome to the reference folder for **Luana**! 

This character was built inspired by the look of my psychologist. I had made this drawing especially to show her during one of our therapy sessions. Because of that, the character doesn't have any initial development.

This character almost became lost media because I accidentally deleted the Paint Tool SAI file where she was stored. However, I got really lucky that a photo of my computer monitor was found—and that is exactly the photo I used to show her how the character's design was coming along.

## 🎨 About the Design
This character is completely original and was 100% designed and created by me.
